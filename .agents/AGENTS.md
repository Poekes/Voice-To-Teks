# Voice-To-Teks Agent Guidelines

Berikut adalah panduan (guidelines) wajib yang harus diikuti oleh agen saat bekerja pada repositori ini:

1. **Commit Per File**: Setiap kali ada perubahan kode atau penambahan file baru, agen WAJIB melakukan `git commit` secara terpisah untuk masing-masing file. (Satu file = satu pesan commit yang deskriptif).
2. **Auto-Update README**: Setiap kali agen menambahkan fitur baru, memodifikasi logika secara signifikan, atau mengubah antarmuka pengguna, agen WAJIB memperbarui file `README.md` (terutama pada bagian tabel fitur) agar dokumentasi selalu sinkron dengan kondisi aplikasi yang terbaru.
